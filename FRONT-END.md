# 前端页面开发

前台系统包括下面的部分

| 页面       | 作用                                                     | 访问限制 |
| ---------- | -------------------------------------------------------- | -------- |
| 首页       | 轮播图，商品分类导航，商品推荐，用户登录，进入购物车页面 | 无       |
| 购物车页   | 查看放入购物车的商品                                     | 需要登录 |
| 分类商品页 | 从分类商品中查找想要的产品                               | 无       |
| 商品详情页 | 商品的详情，包括商品图片，参数，sku，评论等              | 无       |
| 登录页面   | 用户登录                                                 | 无       |
| 个人主页   | 设置个人信息等                                           | 需要登录 |
